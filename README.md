---# 📚 Estudos em Teste de Software

Este repositório documenta meus estudos sobre **Teste de Software e Qualidade**, abordando fundamentos teóricos, terminologia técnica, processos e exemplos práticos.

O objetivo é **consolidar conhecimento**, **organizar o aprendizado** e **aplicar os conceitos de forma estruturada** ao longo das aulas.

---

# 🎯 Objetivos de Aprendizado

Ao final deste estudo, pretendo ser capaz de:

- Compreender o papel do teste no Ciclo de Vida do Software
- Diferenciar **Erro, Defeito e Falha**
- Entender os principais **níveis e tipos de teste**
- Estruturar **casos de teste**
- Elaborar **registros de bugs (bug reports)**
- Desenvolver visão crítica sobre **qualidade de software**
